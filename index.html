About the Project

The operating system for
high-compliance hospitality.

V.A.K is a vertical Workforce Management Operating System built specifically for the low-margin, high-compliance restaurant industry. Built on a hybrid serverless microservices architecture, it enforces strict financial compliance while delivering a fluid, mobile-first experience.

Built With
React Native Expo Supabase .NET 8 Turborepo NativeWind Google Gemini Zod
Geo-Fenced Attendance
GPS-verified clock-ins with a <50m proximity rule. No buddy punching, ever.
Live Schedule via WebSockets
Zero-latency real-time shift publishing. Every device updates the instant a manager saves.
8/44 Compliance Engine
Automated overtime calculation via .NET 8 - full Alberta Labour Standards adherence built in.
P2P Shift Marketplace
Employees swap shifts directly. Manager approval queues keep control in the right hands.

Security & Compliance
Row Level Security - All DB access scoped to restaurant_id at the Postgres level.
Shared Types Contract - Zod schemas keep frontend & backend in sync, always.
PIPEDA Compliant - All PII & labour data stored in Canada (Central).
Protected Main - Branch rules enforced. No direct pushes to main.
